## 油猴脚本

### 程序员就该用程序解决问题

掘金抽奖签到脚本，抽奖可视化
使用petite-vue做的可视化，api可参考使用petite-vue的文档。

功能：
- [x] 签到并自动免费抽奖
- [x] 5连抽
- [x] 10连抽
- [x] 梭哈抽奖

使用方法：
1. 前往 [安装地址](https://greasyfork.org/zh-CN/scripts/431740-%E6%8E%98%E9%87%91%E6%8A%BD%E5%A5%96) 安装使用
2. 打开油猴新建油猴脚本，复制 Juejin_Helper.js 内容到油猴编辑器内，打开掘金即可看到打开抽奖脚本弹窗的按钮。
